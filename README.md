1. Κύρια Συνάρτηση (Main)

Αρχικά, ανοίγεται το αρχείο commands.txt για ανάγνωση εντολών και το αρχείο output.txt για την εγγραφή των αποτελεσμάτων.

Στη συνέχεια:

Δηλώνεται μεταβλητή t για τη χρονομέτρηση εκτέλεσης.
Χρησιμοποιούνται flags (flMIN, flMAX, flGR, flTREE, flHASH) ώστε να ελέγχεται αν έχει προηγηθεί η εντολή BUILD για κάθε δομή.

Επιπλέον:

Οι ακέραιες μεταβλητές num, num2, num3 αρχικοποιούνται σε -1.
Οι συμβολοσειρές c, d, f αποθηκεύουν:
την εντολή,
τη δομή δεδομένων,
το όνομα αρχείου εισόδου αντίστοιχα.
Διαδικασία Ανάγνωσης Εντολών

Το πρόγραμμα διαβάζει διαδοχικά εντολές από το commands.txt:

BUILD → αποθηκεύεται το όνομα αρχείου στο f
SEARCH / INSERT → διαβάζεται ένας ακέραιος (num)
στον γράφο: διαβάζονται τρεις τιμές (num, num2, num3)
COMPUTETHESHORTESTPATH → διαβάζονται δύο ακέραιοι

Η διαδικασία συνεχίζεται μέχρι το τέλος του αρχείου.

Εκτέλεση Εντολών

Με χρήση if - else if:

Αν η εντολή είναι BUILD → ενεργοποιείται το αντίστοιχο flag
Αν είναι άλλη εντολή:
ελέγχεται αν έχει προηγηθεί BUILD
αν ναι → εκτελείται
αν όχι → αγνοείται
